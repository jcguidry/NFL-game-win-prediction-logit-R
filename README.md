# NFL-game-win-prediction-logit-R

 Predictive model for NFL game outcomes - void of data leakage

 

Requirements:

- R
- Pandoc (to render PDFs)

Pandoc Installation:

- MacOS/Linux
  - Install Homebrew
  - `brew update`
  - `brew install pandoc`
  - `sudo pacman -S pandoc`
